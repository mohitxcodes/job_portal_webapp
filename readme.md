# JobFinder

A pure HTML job portal template (No CSS, No JS).

## Features

- **Pure HTML Structure:** Built entirely using HTML4/5 tags and attributes.
- **Table-Based Layout:** Classic layout method ensuring structure without external stylesheets.
- **Sections:**
  - Header & Navigation
  - Job Search (Hero Section)
  - Featured Jobs & Top Recruiters
  - Premium Services & Success Stories
  - Footer & Quick Links

## How to Run

Simply open `index.html` in your web browser.
